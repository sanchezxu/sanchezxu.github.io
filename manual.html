AI小说创作平台 - 操作手册

基于 FastAPI 的 AI 小说创作平台后端服务

项目简介

这是一个基于 FastAPI 的 AI 小说创作平台后端服务，支持：

多AI大模型支持

DeepSeek、Qwen、百川、智谱、豆包、文心等

小说大纲自动生成

智能生成小说结构框架

角色与世界观管理

丰富的角色创建与设定功能

自动写作与章节生成

智能续写与章节创作

写作统计与目标追踪

详细的写作数据分析

多格式导出

TXT、JSON、HTML格式支持

环境要求

  • Python 3.9+
  • Windows / Linux / macOS

安装步骤

1 创建虚拟环境（推荐） 

cd d:\PyProject\novel\Apply
python -m venv venv

2 激活虚拟环境

Windows:

venv\Scripts\activate

Linux/macOS:

source venv/bin/activate

3 安装依赖 

cd d:\PyProject\novel\Apply
pip install -r requirements.txt

运行项目

方式一：使用启动脚本（推荐）

cd d:\PyProject\novel\Apply
python main.py

方式二：使用 uvicorn 直接启动

cd d:\PyProject\novel\Apply
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

访问服务

启动成功后，打开浏览器访问：

地址 说明
http://localhost:8000 应用首页
http://localhost:8000/docs Swagger UI - API交互文档
http://localhost:8000/redoc ReDoc - API参考文档

快速使用指南

1 注册账号 

POST /users/register
{
  "email": "your_email@example.com",
  "password": "your_password"
}

2 登录获取Token 

POST /users/login
{
  "email": "your_email@example.com",
  "password": "your_password"
}

返回的Token使用方式： 在请求头中添加：Authorization: Bearer <your_token>

3 配置API密钥（重要！） 

PUT /users/me
{
  "api_key": "your-llm-api-key",
  "llm_provider": "deepseek",
  "llm_model": "deepseek-chat"
}

4 创建小说 

POST /novels/
{
  "name": "我的小说",
  "genre": "仙侠",
  "target_chapters": 100
}

5 生成大纲与角色 

POST /novels/{novel_id}/outlines/generate
POST /novels/{novel_id}/characters/generate
{
  "count": 5
}

6 启动自动写作 

POST /novels/{novel_id}/auto-write
{
  "start_chapter": 1,
  "end_chapter": 50
}

常见问题

Q1: 启动时报错 "email-validator is not installed"

pip install email-validator

Q2: 数据库报错

删除 novel_apply.db 文件后重新启动，应用会自动创建新数据库。

Q3: API密钥配置后不生效

确保在请求头中正确添加了 Authorization: Bearer <token>

Q4: 端口被占用

修改 app/core/settings.py 中的端口号，或结束占用端口的进程。

注意事项

  • API密钥安全：所有API密钥存储在用户个人账户中，不会泄露
  • 数据存储：小说章节默认保存到 D:/novel 目录
  • Token配额：可在个人设置中调整每日Token限制
  • 自动写作：任务在后台运行，可通过 /tasks 查看进度
